# Compatibility pointer — Identity continuity

This path is retained so historical links do not break.

Identity continuity now has explicit roadmap placement as **`I1`**, a conditional gate in the authoritative [Identity continuity track](../identity/README.md).

`I1` activates only for a proven cross-installation requirement that local export/import cannot solve. It does not create default accounts, does not turn `installation_id` into `person_id`, and forbids IP/MAC/hardware/browser fingerprinting.

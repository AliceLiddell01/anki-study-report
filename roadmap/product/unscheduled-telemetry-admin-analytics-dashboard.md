# Compatibility pointer — Telemetry Admin Analytics Dashboard

This path is retained so historical links do not break.

The internal telemetry dashboard now has explicit roadmap placement as **`O1`** in the authoritative [Telemetry operations track](../operations/README.md).

`O1` is a separate protected read-only application. It does not add an add-on route, generic query API, arbitrary SQL or local admin secrets. Cloudflare Access, Worker-side JWT validation, server-side D1 bindings and prepared bounded queries are required.
